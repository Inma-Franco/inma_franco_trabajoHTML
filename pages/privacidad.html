Política de Privacidad de Zahara

  1. Responsable del Tratamiento de Datos

    2. La empresa Zahara con domicilio en Calle San Luis, 45, 41001, Sevilla, España, es la responsable de los datos personales facilitados por los usuarios. Nos comprometemos a garantizar que la información personal proporcionada esté protegida de acuerdo con la normativa aplicable de protección de datos.

  2. Datos Recopilados

    3. Los datos que recopilamos incluyen:

    • Información personal: nombre, correo electrónico, dirección, número de teléfono, entre otros, que el usuario proporciona al registrarse o al realizar una compra.
    • Datos de navegación: información sobre el uso de nuestra web a través de cookies, como las páginas visitadas y el tiempo de permanencia.

  3. Finalidad del Tratamiento

    4. Utilizamos los datos personales para las siguientes finalidades:

    • Procesar pedidos y gestionar la entrega de productos.
    • Proporcionar atención al cliente.
    • Enviar comunicaciones comerciales relacionadas con productos, promociones y ofertas, siempre que se haya dado el consentimiento explícito.

  4. Base Legal para el Tratamiento

    5. El tratamiento de los datos personales se realiza sobre las siguientes bases legales:

    • El consentimiento del usuario.
    • La ejecución de un contrato para la venta de productos.
    • El cumplimiento de obligaciones legales.

  5. Derechos del Usuario

    6. El usuario tiene los siguientes derechos:

    • Acceso a sus datos personales.
    • Rectificación de información incorrecta o desactualizada.
    • Supresión de los datos cuando ya no sean necesarios para la finalidad para la que fueron recopilados.
    • Oposición al tratamiento de sus datos con fines comerciales.
    • Portabilidad de sus datos a otro responsable.

  6. Compartición de Datos

    7. Zahara no comparte los datos personales con terceros, salvo cuando sea necesario para cumplir con las obligaciones legales o para la prestación de servicios contratados, como el envío de productos.

  7. Conservación de Datos

    8. Los datos personales se conservarán durante el tiempo necesario para cumplir con las finalidades para las que fueron recopilados o para cumplir con las obligaciones legales.
